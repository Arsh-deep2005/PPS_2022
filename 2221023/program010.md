## Program 10: Write a code using break statement.
```
#include<stdio.h>
int main()
{
int i=17;
while(i<30)
{
printf("%d\n",i);
i++;
if(i>25)
{
break;
}
}
return 0;
}
```
**Output**: 
```
           17
           18
           19
           20
           21
           22
           23
           24
           25
```
